# 实习信息智能助手 — 移动端 H5 设计系统规范

> **版本**: 1.1.0（统一版：与 PC 端 `#5b5bd6` + zinc 体系对齐）
> **日期**: 2026-07-19
> **风格**: 简约专业（Linear / Notion / Stripe 启发）
> **载体**: 移动端 H5（375px 基准，最大 480px 居中）
> **无障碍**: WCAG 2.1 AA 合规
> **配套文件**: `design-tokens.css`（可直接引入的 Token）/ `preview.html`（可视化原型）

---

## 一、设计原则

| 原则 | 说明 | 在本项目中的体现 |
|------|------|-----------------|
| **移动优先** | 所有交互为手指触控设计，非 PC 端缩放 | 48px 触摸目标、底部固定操作栏、单手可达 |
| **信息分层** | 核心信息一眼可见，次要信息按需展开 | 岗位卡片：日薪/公司首屏可见，详情点击展开 |
| **克制留白** | 用空间而非线条分隔信息，降低视觉噪音 | 卡片间距 16px、段落间距 24px、减少边框使用 |
| **一致语言** | 相同交互模式贯穿全应用，降低学习成本 | 所有列表项左滑操作一致、所有弹窗从底部滑入 |
| **反馈即时** | 每次操作 150ms 内有视觉反馈 | 按压态 0.7 透明度、Toast 250ms 入场、骨架屏 |

### 2.2 字体系统

```
字体栈: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC",
        "Microsoft YaHei", "Helvetica Neue", sans-serif
等宽:   "SF Mono", "Cascadia Code", "JetBrains Mono", Consolas, monospace
```

**字号阶梯（移动端正文 15px，比 PC 14px 稍大，阅读距离补偿）**

| Token | 字号 | 字重 | 行高 | 用途 |
|-------|------|------|------|------|
| `--fs-xs`  | 12px | 500 | 1.25 | 标签 / 辅助说明 |
| `--fs-sm`  | 13px | 400 | 1.6  | 次要文字 |
| `--fs-base`| **15px** | 400 | 1.6  | **正文（基准）** |
| `--fs-lg`  | 17px | 600 | 1.4  | 卡片标题、导航栏标题 |
| `--fs-xl`  | 20px | 600 | 1.25 | 区块标题 |
| `--fs-2xl` | 24px | 600 | 1.25 | 页面标题 |
| `--fs-3xl` | 30px | 700 | 1.25 | 展示标题 / 大数字 |

> **为什么移动端正文 15px 而非 14px？** 移动设备阅读距离约 30–40cm，比 PC 远；15px 在 375px 屏上等效 PC 14px 的视觉密度，同时保证中文字符清晰可读。

### 2.3 间距与圆角

**间距（4px 基准，8 点网格，与 PC 端统一）**

```
--space-1: 4px   --space-2: 8px   --space-3: 12px  --space-4: 16px
--space-5: 20px  --space-6: 24px  --space-8: 32px  --space-10: 40px
--space-12: 48px --space-16: 64px
```

**使用约定**：组件内 padding 12px（紧凑）/ 16px（标准）；卡片间 gap 12px（列表内）/ 16px（区块间）；页面水平 padding 16px（375px）/ 20px（≥414px）；段落垂直间距 24px。

**圆角（移动端略大于 PC 端，触摸设备更友好）**

| Token | 值 | 用途 |
|-------|----|------|
| `--radius-xs` | 6px | 小标签（PC 端 4px） |
| `--radius-sm` | 8px | 按钮 / 输入框（PC 端 6px） |
| `--radius-md` | 12px | 卡片（PC 端 10px） |
| `--radius-lg` | 16px | 大卡片 / 容器（PC 端 14px） |
| `--radius-xl` | 20px | 弹窗 / 抽屉 |
| `--radius-full` | 9999px | 圆形 / 胶囊 |

### 2.4 阴影系统（zinc 基底，移动端克制不宜过重）

| Token | 值 | 用途 |
|-------|----|------|
| `--shadow-xs` | `0 1px 2px rgba(24,24,27,0.04)` | 卡片静态 |
| `--shadow-sm` | `0 1px 3px rgba(24,24,27,0.06), 0 1px 2px rgba(24,24,27,0.04)` | 卡片悬停 |
| `--shadow-md` | `0 4px 12px rgba(24,24,27,0.08)` | 浮层 / 下拉 |
| `--shadow-lg` | `0 8px 24px rgba(24,24,27,0.10)` | 弹窗 |
| `--shadow-xl` | `0 16px 40px rgba(24,24,27,0.12)` | 大弹窗 |
| `--shadow-brand` | `0 4px 12px rgba(91,91,214,0.25)` | 主按钮专用（主色阴影） |

### 2.5 动效系统

| Token | 值 | 用途 |
|-------|----|------|
| `--duration-fast` | 150ms | 微交互：悬停 / 按压 |
| `--duration-normal` | 250ms | 常规：展开 / 折叠 / Toast 入场 |
| `--duration-slow` | 300ms | 页面转场 |
| `--ease-standard` | `cubic-bezier(0.4,0,0.2,1)` | 标准入场 |
| `--ease-emphasized` | `cubic-bezier(0.2,0,0,1)` | 强调入场（弹窗） |

> 全局尊重 `prefers-reduced-motion`：用户开启「减少动态效果」时，所有动画时长降至 0.01ms（见 `design-tokens.css` 末尾）。

## 三、移动端专属规范

### 3.1 安全区域（刘海屏 / Home Indicator）

```css
padding-top: var(--safe-area-top);      /* 顶部导航栏避让状态栏 */
padding-bottom: var(--safe-area-bottom); /* 底部固定栏避让 Home Indicator */
```

### 3.2 触摸目标（WCAG 2.1 SC 2.5.5 要求最小 44px）

| 场景 | 尺寸 | Token | 示例 |
|------|------|-------|------|
| 紧凑场景 | 40×40px | `--touch-target-sm` | 列表内图标按钮（需配合 ≥8px 间距） |
| 标准交互 | 48×48px | `--touch-target-base` | Tab 栏图标、普通按钮 |
| 主操作 | 56px 高 | `--touch-target-lg` | 底部主按钮、发送按钮 |

### 3.3 容器与固定栏

```css
.app-container {
  max-width: var(--container-max-width); /* 480px, 防平板/桌面过宽 */
  margin: 0 auto;
  min-height: 100vh;
  background: var(--color-bg-page);
}
```

| 变量 | 值 | 说明 |
|------|----|------|
| `--topbar-height` | 56px | 顶部导航栏高度（不含安全区） |
| `--tabbar-height` | `calc(56px + safe-area-bottom)` | 底部 Tab 栏（含安全区） |
| `--inputbar-height` | `calc(60px + safe-area-bottom)` | 对话页底部输入栏（含安全区） |
| `--content-padding-bottom-tabbar` | `calc(tabbar + 16px)` | 列表页内容底部留白 |
| `--content-padding-bottom-inputbar` | `calc(inputbar + 16px)` | 对话页内容底部留白 |

## 四、组件库规范

### 4.1 顶部导航栏（TopAppBar）

固定在屏幕顶部，避让状态栏。

```
┌─────────────────────────────────┐
│  [safe-area-top]                │
│  ←  页面标题              ⋯    │  56px
└─────────────────────────────────┘
```

- **高度**：`56px + safe-area-top`
- **背景**：`--color-bg-surface`
- **底边框**：`0.5px solid var(--color-border)`
- **标题**：17px / 600 / `--color-text-primary`，居中或左对齐
- **返回按钮**：48×48px 触摸区，内含 24px 箭头图标
- **右侧操作**：最多 2 个，每个 48×48px
- **变体**：默认（白底+返回箭头）/ 主页（Logo+标题+操作）/ 沉浸式（透明背景，滚动渐显，用于 AI 对话页）

### 4.2 底部 Tab 栏（BottomTabBar）

移动端核心导航，固定在屏幕底部。

```
┌─────────────────────────────────┐
│   📋          💬          👤     │
│   岗位        对话        我的   │  56px
│  [safe-area-bottom]             │
└─────────────────────────────────┘
```

- **高度**：`56px + safe-area-bottom`
- **背景**：`--color-bg-surface`；**顶边框**：`0.5px solid var(--color-border)`
- **3 个 Tab 等分**；图标 24×24px，触摸区 48×48px；文字 11px / 500
- **未选中**：`--color-text-tertiary`；**选中**：`--c-primary-500`（图标填充态）
- **切换动效**：颜色 150ms 过渡，无位移

| Tab | 图标 | 页面 | 说明 |
|-----|------|------|------|
| 岗位 | 列表 | 岗位列表 | 默认首页，含导入 / 筛选 |
| 对话 | 气泡 | AI 问答 | 智能问答主入口 |
| 我的 | 用户 | 个人中心 | 求职画像 / 收藏 / 设置 |

### 4.3 按钮（Button）

| 变体 | 背景 | 文字 | 边框 | 其他 |
|------|------|------|------|------|
| **Primary 主按钮** | `--c-primary-500` | `#fff` 15px/600 | 无 | 高 48px（主操作 56px）；hover `--c-primary-600`；按压 opacity 0.7；底部主操作用 `--shadow-brand` |
| **Secondary 次按钮** | `--color-bg-surface` | `--c-primary-500` | `1px solid --c-primary-200` | 其余同主按钮 |
| **Ghost 幽灵按钮** | 透明 | `--color-text-secondary` | 无 | 按压背景 `--color-bg-muted` |
| **IconButton 图标按钮** | 透明 | — | 无 | 48×48px 触摸区，内含 24px 图标；按压 `--color-bg-muted`（12px 圆角） |

- **圆角**：`--radius-sm`（8px）；**内边距**：水平 24px
- **禁用态**：背景 `--c-gray-200`，文字 `--c-gray-400`，`pointer-events: none`

### 4.4 输入框（TextInput）

```
┌─────────────────────────────────┐
│ 请输入岗位关键词...             │  48px
└─────────────────────────────────┘
```

- **高度**：48px；**背景**：`--color-bg-surface`；**圆角**：`--radius-sm`
- **边框**：`1px solid --color-border`；聚焦变 `--c-primary-500`，外加 `0 0 0 3px var(--c-primary-50)` 光环
- **文字**：15px / `--color-text-primary`；**占位符**：15px / `--color-text-tertiary`
- **错误态**：边框 `--c-error-500`，下方 12px 红色提示文字（`--c-error-700`）
- **多行（Textarea）**：最小高 96px，`resize: none`（移动端不可拖拽）

### 4.5 卡片（Card）

**标准卡片**：背景 `--color-bg-surface`、圆角 `--radius-md`（12px）、内边距 16px、无边框（用 `--shadow-xs` 区分层次）；按压 `--shadow-sm` + 背景 `--c-gray-50`（轻微）。

**岗位卡片（JobCard）— 核心业务组件**

│  [¥180/天] [双休] [6个月]      │  ← 标签组
│  [观山湖]  [免费午餐]            │
│                                 │
│  要求: 本科在读, 了解大模型      │  ← 要求（可折叠）
│  ─────────────────────────       │
│  💬 问AI  📍 导航  📋 详情      │  ← 操作区

- 岗位名 15px/600/`--color-text-primary`；公司名 13px/400/`--color-text-secondary`

### 4.7 对话气泡（ChatBubble）

```
    ┌──────────────────┐
    │ 帮我找日薪150以上的岗位  │  ← AI 气泡（左）
    └──────────────────┘
  🤖

                  │ 哪些岗位双休      │  ← 用户气泡（右）

```

- **AI 气泡**：背景 `--color-bg-surface`、边框 `0.5px solid --color-border`、圆角 16px（左上角 4px）、文字 15px/400、内边距 12px 16px、最大宽 75% 屏宽、头像 32px 圆形（`--c-primary-50` 底 + emoji）

- **Markdown 渲染**：表格横向滚动（表头 `--color-bg-muted`）；代码块 `--c-gray-900` 底 `--c-gray-100` 字 13px mono；列表缩进 20px

### 4.8 底部输入栏（ChatInputBar）

对话页固定栏。

```
┌─────────────────────────────────┐
│ [快捷 ▼]  输入...        [发送] │  60px
│ [safe-area-bottom]              │
└─────────────────────────────────┘
```

- **高度**：`60px + safe-area-bottom`；**顶边框**：`0.5px solid --color-border`
- 快捷问题入口：左侧 32px 按钮，点击展开抽屉
- 输入框：flex-1，高 40px，圆角 full
- 发送按钮：40×40px，`--c-primary-500` 底 + 发送图标；禁用态 `--c-gray-200`

### 4.9 Toast 提示

- **位置**：屏幕底部距底 80px（避开输入栏）；**背景**：`--c-gray-900`；**文字**：`#fff` 14px/500
- **圆角**：`--radius-sm`；**内边距**：12px 24px；**阴影**：`--shadow-lg`
- **入场**：250ms 从下 20px 淡入；**持续**：2.5s 自动消失；**最多 1 个**（新的替换旧的）
- **变体**：默认（无图标）/ 成功（✓ 圆圈）/ 错误（✕ 圆圈）

### 4.10 骨架屏（Skeleton）

列表 / 卡片加载态，替代 spinner。

```css
.skeleton {
  background: linear-gradient(90deg,
    var(--c-gray-100) 0%, var(--c-gray-200) 50%, var(--c-gray-100) 100%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}
@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

- 基底 `--c-gray-100`、闪烁区 `--c-gray-200`、1.5s 循环；圆角与目标组件一致（卡片 12px、文字行 4px）

### 4.11 空状态（EmptyState）

```
┌─────────────────────────────────┐
│         [插图 64px]             │
│        还没有岗位数据            │
│   点击下方按钮导入示例岗位       │
│       [ 导入示例岗位 ]           │
└─────────────────────────────────┘
```

- 插图 64px 线性图标 `--c-gray-300`；主文字 15px/500/`--color-text-secondary`；副文字 13px/400/`--color-text-tertiary`；按钮次按钮居中；垂直居中于可用区域

### 4.12 筛选栏（FilterBar）

岗位列表顶部，横向滚动。

```
┌─────────────────────────────────┐
│ [全部] [符合画像] [收藏] [日薪▼] │  ← 可横滚
└─────────────────────────────────┘
```

- 高度 40px；筛选项胶囊形高 28px 圆角 full；未选中 `--color-bg-muted` 底 + `--color-text-secondary` 字；选中 `--c-primary-500` 底 + 白字；间距 8px；横向滚动隐藏滚动条，右侧 20px 渐变遮罩

### 4.13 抽屉 / 弹窗（BottomSheet / Modal）

从底部滑入的临时面板（快捷问题、筛选详情）。

- **遮罩**：`rgba(0,0,0,0.4)`，点击关闭
- **面板**：背景 `--color-bg-surface`、圆角 `--radius-xl`（顶部 20px）、从底部滑入 250ms `--ease-emphasized`
- **拖拽条**：顶部 4px×36px 圆角条 `--c-gray-300` 居中
- **内容区**：最大高 80vh，内部可滚动

## 五、信息架构与页面流转

### 5.1 底部 Tab 分组

| Tab | 页面 | 核心模块 |
|-----|------|---------|
| 岗位 | 岗位列表 | 导入栏、筛选栏、岗位卡片列表、空状态 |

### 5.2 核心页面（详见 `preview.html` 原型）

1. **岗位列表页**：顶部栏（标题+导入）→ 筛选栏 → 岗位卡片列表（左滑可操作：收藏/删除）→ 空状态（无数据时）
2. **AI 对话页**：沉浸式顶栏 → 对话流（气泡）→ 底部固定输入栏（快捷问题+输入框+发送）
3. **个人中心页**：头像/昵称 → 求职画像卡片（稳定性/细致/坐班能力标签）→ 收藏列表入口 → 设置分组

### 5.3 核心交互模式

| 模式 | 实现 |
|------|------|
| 固定输入栏 | 对话页底部 `position: fixed`，内容区 `padding-bottom: var(--content-padding-bottom-inputbar)` |
| 下拉刷新 | 列表顶部下拉触发，显示品牌色 spinner（尊重 reduced-motion） |
| 左滑操作 | 岗位卡片左滑露出「收藏 / 删除」，统一手势 |
| 底部弹窗 | 所有临时面板从底部滑入，遮罩点击关闭 |
| 收藏即时反馈 | 星标激活 + Toast「已收藏」 |

## 六、无障碍规范（WCAG 2.1 AA）

| 维度 | 要求 | 本系统实现 |
|------|------|-----------|
| 颜色对比 | 普通文字 ≥4.5:1，大文字 ≥3:1 | 主文字 17.7:1、次文字 7.7:1、主按钮 5.2:1，全部达标 |
| 键盘导航 | 全功能无需鼠标 | `:focus-visible` 2px 主色描边；逻辑 Tab 顺序 |
| 屏幕阅读器 | 语义 HTML + ARIA | 导航用 `<nav>`、按钮用 `<button>`、图标加 `aria-label` |
| 触摸目标 | 最小 44px | 默认 48px，紧凑场景也不低于 40px |
| 动效敏感 | 尊重用户偏好 | 全局 `prefers-reduced-motion` 降级 |
| 文字缩放 | 支持 200% 缩放 | 字号用 `rem`/`px` 固定值 + 流式布局，不截断 |
| 错误预防 | 清晰标签与校验 | 输入框聚焦光环、错误态红框+提示文字 |

## 七、与 PC 端的关系（统一点 vs 平台差异）

**统一（同一套视觉语言，无割裂感）**：

| 维度 | 值 |
|------|----|
| 主色 | `#5b5bd6`（50/200/600/700 对齐 PC 端 light/border/hover/active） |
| 中性色 | zinc 暖灰（`#18181b` / `#52525b` / `#e4e4e7` 与 PC 端一致） |
| 语义色 | 降饱和度版（`#16a34a` / `#dc2626` / `#0891b2`） |
| 阴影基底 | `rgba(24,24,27,…)`（zinc） |
| 暗色主题 | 对齐 PC 端 `[data-theme="dark"]`（zinc 深色体系） |

**合理的平台差异（移动端适配，非不一致）**：

| 维度 | PC 端 | 移动端 | 原因 |
|------|-------|--------|------|
| 正文字号 | 14px | 15px | 移动阅读距离更远 |
| 圆角 | 4/6/10/14px | 6/8/12/16px | 触摸设备更友好 |
| 导航 | 左侧边栏 | 底部 Tab 栏 | 单手操作 |
| 主操作 | 行内按钮 | 底部固定栏 | 移动端拇指热区 |

### 8.3 改造流程（8 步）
