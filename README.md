# CareerMatch AI - 求职智配平台

> 从岗位筛选到简历优化、面试模拟、职业规划的全流程 AI 求职助手。纯前端实现，零后端零部署。

## 四大功能模块

### 1. 岗位匹配（智能筛选）
- CSV / 自由文本导入岗位数据
- **7 维匹配度评分**（0-100分）：地点/薪资/工作制/时长/餐饮/住宿/岗位方向
- 匹配度环形图可视化，点击查看分维度评分明细
- 排序：按匹配度 / 日薪高到低 / 日薪低到高
- 筛选：全部 / 符合画像 / 收藏
- 求职画像可编辑，修改后自动重新计算匹配度
- AI 智能问答：基于岗位数据回答推荐/对比/排序问题

### 2. 简历优化
- 粘贴简历 + 目标岗位 JD
- AI 分析匹配度（分维度打分）
- 找出缺失关键词
- 用 STAR 法则优化经历描述
- 给出可直接使用的修改建议

### 3. 面试模拟
- 选择岗位 + 面试类型（综合/技术/行为/HR）
- AI 扮演面试官，逐轮提问 + 追问
- 由浅入深：自我介绍 → 项目经历 → 专业能力 → 行为问题
- 面试结束给出综合评价和改进建议

### 4. 职业规划
- 填写专业/学历/毕业年份/意向方向/城市
- AI 生成完整职业规划报告：
  - 适合的职业方向分析（3+方向）
  - 短期行动路径（3-6个月）
  - 中长期发展规划（1-3年）
  - 技能提升建议 + 学习资源
  - Plan B 备选方案

## UI 特性
- 暗色 / 亮色模式切换
- 响应式布局（移动端适配）
- 匹配度环形图 + 分维度进度条
- Tab 导航

## 快速开始

```bash
cd intern-assistant
python -m http.server 8084
```

浏览器打开 `http://localhost:8084`

首次使用需配置 DeepSeek API Key（右上角 → Key）。

## 技术栈
- Vanilla HTML / CSS / JS
- DeepSeek API（OpenAI 兼容格式）
- Chart.js（雷达图）
- marked.js（Markdown 渲染）
- localStorage（画像/收藏/主题持久化）
